# 更新日志

本文档记录 `Qproxyhub` 的公开版本变更。

---

## v0.1.0

首个公开预览版。

### 已包含

- 本地单页面代理测试工作台
- 支持 `socks5`、`http`、`hy2`、`vless`、`vmess`
- 实时进度反馈
- 历史任务记录
- 失败项重试
- 结果排序、筛选、复制
- 导出为 `TXT`、`CSV`、`JSON`
- 导出到 `mihomo`、`Clash Verge`、`v2rayN`、`NekoBox / Nekoray`
- Windows 启动器 `Qproxyhub.exe`
- 便携版压缩包下载

### 说明

- 当前仍是预览阶段
- 推荐优先使用 `portable.zip`
- 对同源代理池测试时，推荐使用：
  - `8 秒` 或 `12 秒` 超时
  - `1 条并发`
